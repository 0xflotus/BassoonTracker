Version 0.3

Most noticable new features

Fasttracker XM support

This includes support for
  • up to 32 channels
  • up to 128 instruments
  • extra instrument properties like volume envelopes, panning envelopes, fadeout, more precise finetuning
  • Extra effects like global volume

Drop box integration

load and save your files directly from and to Dropbox. The first time you active this feature, tou will have connect Bassoontracker to Dropbox and allow it access ypur files. Bassoontracker will only have access to it own files in the "Apps/Bassoontracker" folder, not to your entire Dropbox.

Export entire song to .wav or .mp3

Create a high quality mixdown of oyour song to .wav or .mp3.
This feature requires an internet connection.
Your file is uploaded to the Bassoon server and the conversion is done serverside.

Select/cut/copy/paste ranges of a pattern

You can select a range in recording mode by just dragging over the pattern or by shift-dragging in any mode. Ranges can include multiple colums. A edit menu will appear when the range is selected.
